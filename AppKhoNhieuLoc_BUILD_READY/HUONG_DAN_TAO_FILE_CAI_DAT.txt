HƯỚNG DẪN TẠO FILE CÀI ĐẶT APP KHO NHIÊU LỘC
================================================

BỘ ZIP NÀY LÀ GÌ?
-----------------
Đây là bộ source app Electron đã được dọn sạch và thêm cấu hình build installer.
Dùng bộ này để tạo file cài đặt .exe cho các máy tính khác.

CÁC FILE CHÍNH
--------------
main.js
preload.js
package.json
LenhKho_Template.xlsx
PhieuXuat_Template.xlsx
Bill_Template.xlsm
ChuyenKho_Template.xlsx

CÁCH TẠO FILE CÀI ĐẶT TRÊN MÁY CỦA BẠN
---------------------------------------
1. Giải nén thư mục này ra máy.

2. Cài Node.js LTS nếu máy chưa có.
   Link tải: https://nodejs.org

3. Bấm chạy:
   01_CAI_THU_VIEN.bat

   Bước này cần Internet và chỉ cần chạy lần đầu.
   Nó sẽ tải Electron + electron-builder vào thư mục node_modules.

4. Bấm chạy thử app:
   02_CHAY_THU_APP.bat

5. Nếu chạy ổn, bấm tạo file cài đặt:
   03_TAO_FILE_CAI_DAT_EXE.bat

6. Sau khi build xong, vào thư mục:
   dist

   Bạn sẽ thấy file dạng:
   AppKhoNhieuLoc-Setup-1.0.0.exe
   AppKhoNhieuLoc-Portable-1.0.0.exe

CÀI TRÊN MÁY TÍNH KHÁC
----------------------
Chỉ cần copy file trong thư mục dist sang máy khác rồi chạy file Setup .exe.

Máy tính khác KHÔNG cần cài Node.js.
Nhưng máy đó cần:
- Có Internet để mở web app CodeSandbox.
- Có Microsoft Excel bản desktop, vì app in ngầm qua Excel.
- Có driver máy in đã cài trong Windows.

KHI NÀO CẦN BUILD LẠI APP?
--------------------------
Không cần build lại nếu chỉ sửa code web trên CodeSandbox.

Cần build lại nếu bạn sửa:
- main.js
- preload.js
- file mẫu Excel .xlsx / .xlsm
- cấu hình máy in trong Electron

LƯU Ý VỀ CHUYỂN KHO
-------------------
main.js trong bộ này đã mở vùng in Chuyển Kho đến dòng 18:
- CKAB: A1:N18
- CKNB: A1:J18

Nên chữ Kế toán / Xe nâng ở dòng 16 và 18 trong ChuyenKho_Template.xlsx sẽ không bị mất khi in.

Nếu muốn đổi tên app hoặc version, sửa trong package.json:
- productName
- version
